📚 API Documentation

RESTful API endpoints for bot integration

Pairing Endpoints

POST /api/pair/code - Generate pairing code
GET /api/pair/status/:sessionId - Check pairing status
GET /api/pair/sessions - List all sessions
DELETE /api/pair/session/:sessionId - Remove session

Bot Endpoints

POST /api/bot/start - Start bot instance
POST /api/bot/stop - Stop bot instance
GET /api/bot/status - Get bot status
POST /api/bot/send - Send message

Health Endpoints

GET /health - Health check
GET /health/ready - Readiness probe
GET /health/live - Liveness probe
